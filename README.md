## Краткое описание

Данный проект является практической работой, выполняемой в рамках курса [Мидл фронтенд-разработчик](https://praktikum.yandex.ru/middle-frontend/) от [Яндекс.Практикум](https://praktikum.yandex.ru).

## Установка и запуск

### Установка

```bash
npm install
```

### Запуск проекта:

```bash
npm start
```

### Запуск сервера: 

```bash
npm run server
```

### Запуск тестов:

```bash
npm run test
```
